Здраствуйте, моя программа называется "обучашка". Этот проект создан с целью развития детей в сфере математике. 
Мои примеры поготут оттачить счет и со временем ребенок сможет быстрее считать.
В этом проекте я использовала различные виджеты, также мной были добавленны 
картинки, с целью мотивации будущих пользователей.
